# jQuery Watermarking #

We do a couple of exercises with the core topic of "watermarking". We use a jQuery Plugin to render visible watermark (images) on other images. 

## Important Note #
<span style="color:red">You must run the examples from your local webserver (Apache, XAMPP)</span>

To do that, copy the example files to your `htdocs` directory (Usually in `C:\xampp\htdocs\` (Windows) or `/Applications/XAMPP/htdocs/` (macOS)). 

Then open your browser and type `http://localhost/09-jquery-watermarks/breakout-solution/watermark-service.html`. 

Let us know if you need assistance.
